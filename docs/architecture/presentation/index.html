02

How It Works

Two global shutter cameras + IR LED strobes freeze a golf ball in flight. 20-microsecond light pulses capture 3-5 positions in a single exposure. From those positions, we calculate everything.

06

Trait in Action

The vision pipeline doesn't know or care what camera it's using. Pass in a MockHardware for testing, a real LibCamera for production.

// The trait — defines what a camera can do pub trait CameraCapture: Send + Sync { fn capture_strobed(&mut self) -> Result<Vec<ImageFrame>>; } // Production: real camera on Pi/Jetson let mut camera = LibCamera::new(0)?; let frames = camera.capture_strobed()?; // Testing: replay saved images, no hardware needed let mut mock = MockHardware::from_fixture("tests/fixtures/driver_shot_001")?; let frames = mock.capture_strobed()?; // Same pipeline processes both — it only sees the trait fn process_shot(cam: &mut dyn CameraCapture) -> ShotData { let frames = cam.capture_strobed().unwrap(); let pipeline = VisionPipeline::new(); pipeline.process(&frames) }

08

What It Measures

From 3-5 frozen ball positions captured in microseconds, the vision pipeline extracts:

Ball Speed

Distance between positions / known time intervals. Pinhole camera model converts pixels to real-world distance.

Launch Angle

Vertical: trajectory slope in the camera plane. Horizontal: offset from center line using stereo cameras.

Backspin

Gabor filter extracts dimple pattern orientation between frames. Rotation delta / time = RPM.

Sidespin

Same dimple tracking on the horizontal axis. Combined with backspin gives total spin and spin axis.

Get Started

No hardware needed. Clone, build, run the full pipeline on synthetic data.

# Clone and build git clone https://github.com/r87-e/LaunchTrac.git cd LaunchTrac cargo build # Run all tests cargo test --workspace # Generate synthetic ball images cargo run -p launchtrac-edge --example generate_fixtures # Process a fixture through the full pipeline cargo run -p launchtrac-edge --example process_fixture -- tests/fixtures/driver_shot_001 # Start with web dashboard on :8080 cargo run -p launchtrac-edge -- --mock --port 8080
